Acceleration Measurement App

Acceleration Analysis App

This is a web app for analyzing acceleration data measured using the above Acceleration Measurement Web App (or CSV-format data from other acceleration measurement apps). It calculates velocity and position from acceleration data through numerical integration and visualizes the results as graphs. Simply access the above link to use this app (No installation required).
