■ Overview

This application is an acceleration data analysis web app that calculates velocity and position through numerical integration of acceleration data measured by tools such as Acceleration measurement Web App, and visualizes the results as graphs. It features data smoothing (noise removal) and coordinate axis transformation using principal component analysis (PCA). Since it can read acceleration data saved in CSV format, it is capable of analyzing acceleration data measured by various acceleration measurement apps. This is a web application that runs in a web browser and does not require installation.


■ System requirements

  • PC with web browsers
    Available if a Javascript-enabled browser is installed.
    As for OS, Windows, MacOS, or Linux are supported.
    Browsers that have been tested: Chrome, Edge, Firefox, Safari

  • Smart phone / Tablet
    ◆ Android : Available in any version (newer is better)
    Browsers that have been tested: Chrome, Edge, Firefox
    ◆ iPhone / iPad : Available in any version (newer is better)
    Browsers that have been tested: Safari
Library used in app
The following JavaScript open source libraries are used:
math.js (ver.14.0.0): Mathematical function library (Apache 2.0 license). Used for computing various mathematical functions and Fourier transforms.
pca.js (ver.1.0.2): Principal component analysis library (MIT license). Used in coordinate axis transformation by principal component analysis.
Plotly.js (ver.2.35.2): Graphing library (MIT license). Used for graph drawing.
