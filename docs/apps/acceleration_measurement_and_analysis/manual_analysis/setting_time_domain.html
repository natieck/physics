■ Set Time Domain for Analysis

After selecting the time and acceleration columns, review the graph of the selected acceleration data and set the time domain to be used for analysis.


In the graph of acceleration below, if you want to use acceleration data from 1.6 seconds to 4.6 seconds for analysis, select 1.6 as the start time and 4.6 as the end time in the time domain settings.

After selecting the start time and end time, click the "Select" button on the right. A new graph of acceleration data within the specified time domain will be displayed for each component. By clicking "▶ Acceleration Data" above the acceleration graph, a table of acceleration data within the specified time domain will appear (click it again to close the table).

If necessary, perform zero adjustment (shifting all acceleration data to align to zero) or smoothing of acceleration data (using Fourier transformation or least-squares approximation for smoothing). To calculate velocity data from acceleration data using numerical integration, set the initial velocity for each component and click "Calculate Velocity (numerical integration)."
