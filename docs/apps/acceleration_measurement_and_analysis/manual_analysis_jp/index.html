■ 概要

本アプリは，加速度測定Webアプリなどで測定した加速度データから，数値積分により速度・位置を求めてグラフ化する加速度データ解析Webアプリである．データの平滑化（ノイズ除去）機能や主成分分析による座標軸変換機能を搭載している．CSV形式で保存された加速度データを読み込めるので，様々な加速度測定アプリで測定した加速度データを解析できる．Webブラウザ上で動作するWebアプリのためインストールは不要である．


■ 動作環境

  • Webブラウザを搭載したパソコン
    Javascriptを有効にしたブラウザがインストールされていれば利用可能．
    OS は Windows, MacOS, Linux のいずれでも可．
    動作確認済みのブラウザ：Chrome, Edge, Firefox, Safari

  • スマートフォン・タブレット
    ◆ Android：どのバージョンでも利用可能（できるだけ新しい方が良い）．
    動作確認済みのブラウザ：Chrome, Edge, Firefox
    ◆ iPhone・iPad：どのバージョンでも利用可能（できるだけ新しい方が良い）．
    動作確認済みのブラウザ：Safari
【使用ライブラリ】
以下のオープンソースJavaScript用ライブラリを用いている．
math.js (ver.14.0.0)：数学関数ライブラリ（Apache2.0ライセンス）．数学の各種関数の計算やフーリエ変換で使用している．
pca.js (ver.1.0.2)：主成分分析ライブラリ（MITライセンス）．主成分分析による座標軸変換で使用している．
Plotly.js (ver.2.35.2)：グラフ作成ライブラリ（MITライセンス）．グラフ描画で使用している．
